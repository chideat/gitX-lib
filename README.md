GitX lib C++ Interfaces
=======================

This project contributes to the development of github and gitlab Interfaces based on qt.


GitHub API(v3)
---------------
* authorization
* Activity
* Gists
* Git Data
* Issues
* Orgs
* Pull Requests
* repos
* Users
* search 


GitLab API
------------
* Users
* Session
* Projects
* Snippets
* Repos
* Issues
* Milestones
* Note: Wall
